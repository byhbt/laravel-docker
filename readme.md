
## Laravel Docker
- using docker 3.3 syntax
- Adminer supported.

## How to use

Step 1:

Install docker and docker-compose to your pc.

Step 2:

```
git clone
```
```
docker-compose up
```
```bash
chmod -R 777 storage
```
